Dejafeed
========

mini feed reader/writer

Installation
============

  * npm install express feedparser request validator jsdom
  * mkdir data
  * mkdir data/posts
  * node deja.js
  * http://localhost:8000 (or change 1st line of deja.js)

Roadmap
============

  * bookmarklet to subscribe
  * back-end for data
  * front-end
